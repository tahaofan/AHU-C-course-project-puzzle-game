# 🧩 拼图大闯关 - EasyX Puzzle Game

基于 **EasyX 图形库** 开发的一款 Windows 桌面拼图游戏，包含 7 个独立关卡，支持鼠标拖拽拼图、碎片分页滚动、通关弹窗等功能。

## ✨ 功能特性

- **主菜单** — 开始游戏 / 退出游戏（含确认弹窗）
- **关卡选择** — 7 个关卡以图标网格形式展示，点击即玩
- **拼图玩法** — 从底部碎片栏拖拽碎片到拼图区域，释放后自动吸附至最近网格
- **碎片分页** — 12 块碎片分 3 页展示（每页 5 块），左侧竖排按钮翻页
- **自动判定** — 碎片到达正确位置后自动锁定（绿色高亮标记目标位置）
- **通关弹窗** — 拼图完成后弹出提示，可进入下一关或返回主菜单
- **背景音乐** — 主菜单循环播放背景音乐（MP3 格式）
- **关卡重置** — 游戏过程中可随时重新开始当前关卡

## 🎮 操作说明

| 操作 | 描述 |
|------|------|
| **鼠标左键** | 点击按钮 / 拖拽拼图碎片 |
| **翻页按钮** | 左侧竖排圆形按钮（1/2/3），切换碎片页 |
| **重新开始** | 右上角 ↺ 按钮，重置当前关卡 |
| **返回菜单** | 右上角 ☰ 按钮，返回主菜单 |

## 🖼️ 资源说明

所有图片资源位于 `puzzlegame/puzzlegame/assets/` 目录下（从项目根目录）：

- `主菜单封面.jpg` / `选择关卡背景.jpg` — 界面背景
- `开始游戏.jpg` / `退出游戏.jpg` — 主菜单按钮（含悬停状态 `*2.jpg`）
- `关卡1.jpg` ~ `关卡7.jpg` — 关卡完整预览图
- `关卡1碎片/` ~ `关卡7碎片/` — 每关 12 块拼图碎片（160×90 像素，3×4 布局）
- `0.mp3` / `01.mp3` / `02.mp3` — 背景音乐与音效

## 🛠️ 技术栈

- **语言** — C/C++
- **图形库** — [EasyX](https://easyx.cn/) (2024 或更新版本)
- **音频** — Windows MCI (winmm.lib)
- **开发环境** — Visual Studio 2022
- **窗口尺寸** — 1024 × 576 像素

## 🚀 构建与运行

1. 安装 [EasyX 图形库](https://easyx.cn/)
2. 使用 **Visual Studio 2022** 打开 `puzzlegame/puzzlegame.sln`
3. 选择 **x86 (Win32)** 平台（EasyX 仅支持 32 位）
4. 按 `F5` 编译并运行

> ⚠️ 确保 `assets/` 目录与可执行文件在同一目录下（VS 中默认与 `.vcxproj` 同目录）。

## 📝 模块说明

| 模块 | 文件 | 职责 |
|------|------|------|
| Main | `main.cpp/.h` | 窗口初始化、`gameState` 状态机主循环 |
| A_task | `A_task.cpp/.h` | 主菜单绘制、开始/退出按钮、背景音乐 |
| B_task | `B_task.cpp/.h` | 关卡选择界面（4+3 图标布局） |
| C_task | `C_task.cpp/.h` | 拼图碎片数组管理、拖拽检测、网格吸附、完成判定 |
| D_task | `D_task.cpp/.h` | 底部滚动条（3 页分页、翻页按钮、重新开始） |
| E_task | `E_task.cpp/.h` | 通关弹窗绘制与交互、`runPuzzleGame` 主循环 |
| F_task | `F_task.cpp/.h` | IMAGE 全局变量定义、`loadresources` 资源加载、关卡数据 |
